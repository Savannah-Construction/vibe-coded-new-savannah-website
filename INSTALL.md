# Install

Copy the contents of this folder into:

```powershell
C:\CODE\Savannah\Website\savannah-website
```

Then run:

```powershell
git status
git add .
git commit -m "Rebuild website with reviewed image references"
git push
```

Open `index.html` with Live Preview to review before deployment.

This package excludes raw source images, raw camera files, logs, databases and video files. It contains the complete website code plus the images referenced by that code.
