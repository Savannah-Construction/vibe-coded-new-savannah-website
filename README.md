# Savannah Construction Website

Populated static website starter based on the current public Savannah Construction website.

## Included

- Home page
- Projects page
- Team page
- Contact page
- Blog summary page
- Community support placeholder
- Current navigation/footer structure
- Current contact details
- Current project archive
- Current team list
- Current public blog article summaries
- Local image references using files from the supplied zip
- Repo-safe `.gitignore`
- Disabled deployment example

## Recommended local folder

```text
C:\CODE\Savannah\Website\savannah-website
```

## Use

```powershell
git status
git add .
git commit -m "Populate website from current live site"
git push
```

## Important

Images used by the current pages are now stored locally in `assets/img`. See `docs/IMAGE-PLACEMENT.md` and `assets/image-manifest.json`.
